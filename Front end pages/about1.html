Expire Date Reminder

This is the web application which will reminding owner on approaching expire date of food items and other products for effectively expiration date management of food items in supermarkets and wholesale stores.

The user will enter the informations of the products include product name, quantity, batch number and expiration date through scanning by barcode scanner, input CSV file or manual entry of informations. so then the user will customize reminder by set proximity of reminder, frequency of reminder, reminder preference so he will receive reminder notification through email, in-app notification or push notification.

Also through the application user can able to view the expire dates of the products with the dispaly option and can view the details of expiration date
